# Markdown 快速參考卡

> 列印這張卡片，放在桌上隨時參考！

---

## 📝 基本語法

| 效果 | 語法 | 範例 |
|------|------|------|
| **粗體** | `**文字**` | `**重要**` → **重要** |
| *斜體* | `*文字*` | `*強調*` → *強調* |
| ~~刪除線~~ | `~~文字~~` | `~~舊的~~` → ~~舊的~~ |
| `程式碼` | `` `程式碼` `` | `` `code` `` → `code` |

## 📋 標題

```markdown
# 一級標題
## 二級標題
### 三級標題
#### 四級標題
```

## 📝 列表

### 無序列表

```markdown
- 項目 1
- 項目 2
  - 子項目
```

### 有序列表

```markdown
1. 第一項
2. 第二項
3. 第三項
```

## 🔗 連結和圖片

### 連結

```markdown
[連結文字](https://example.com)
```

### 圖片

```markdown
![圖片說明](/img/圖片名稱.jpg)
```

## 📊 表格

```markdown
| 欄位1 | 欄位2 | 欄位3 |
|-------|-------|-------|
| 資料1 | 資料2 | 資料3 |
```

**對齊方式：**
- `:---` = 左對齊
- `:---:` = 置中
- `---:` = 右對齊

## 📏 增加段落間距

Markdown 不支援直接設定間距，需要使用 HTML：

```markdown
文字內容

<div class="spacing-md"></div>

下一段內容
```

或使用 `<br>` 標籤：

```markdown
文字內容

<br><br>

下一段內容
```

間距類別：
- `spacing-xs` - 小
- `spacing-sm` - 中小
- `spacing-md` - 中等
- `spacing-lg` - 大
- `spacing-xl` - 超大

## 💡 常用符號

- `#` = 標題
- `-` = 列表
- `*` = 斜體或列表
- `**` = 粗體
- `` ` `` = 程式碼
- `>` = 引用
- `|` = 表格
- `---` = 分隔線

# 常見問題與詞彙表

## 常見問題

### Q: 如何追蹤貨物？

A: 您可以透過我們的追蹤系統，輸入訂單號碼或運單號碼即可查詢。

### Q: 運送時間需要多久？

A: 運送時間依運輸方式而定，請參考服務項目頁面。

### Q: 如何聯絡客服？

A: 您可以透過 24/7 客服熱線或線上客服與我們聯絡。

## 詞彙表

### 運輸相關

- **FCL (Full Container Load)**: 整櫃運輸
- **LCL (Less than Container Load)**: 併櫃運輸
- **冷鏈物流**: 需要溫控的物流服務

### 服務相關

- **即時追蹤**: 隨時查詢貨物狀態的服務
- **保險保障**: 為貨物提供的保險服務

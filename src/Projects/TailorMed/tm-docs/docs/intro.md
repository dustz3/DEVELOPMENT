# 歡迎使用 TailorMed 文檔

歡迎來到 **TailorMed 專案文檔中心**！

這裡是 TailorMed 專案的完整文檔，包含所有相關的技術文檔、API 說明、使用指南等。

## 開始使用

### 開發環境需求

- [Node.js](https://nodejs.org/en/download/) 版本 20.0 或以上：
  - 安裝 Node.js 時，建議勾選所有與依賴相關的選項。

### 啟動開發伺服器

在 `tm-docs` 目錄下執行：

```bash
npm install
npm start
```

開發伺服器會在 http://localhost:3000/ 啟動，並且會自動重新載入您的變更。

### 建置生產版本

```bash
npm run build
```

建置完成後，可以使用以下命令預覽：

```bash
npm run serve
```
